Orientation

Meshes.OrientationType — Type
OrientationType

The different types of orientation of a ring. Possible values are CW and CCW.

source
Meshes.orientation — Function
orientation(geom, [method])

Returns the orientation of the geometry geom as either counter-clockwise (CCW) or clockwise (CW).

Optionally, specify the orientation method.

See also WindingOrientation, TriangleOrientation.

source
Meshes.OrientationMethod — Type
OrientationMethod

A method for finding the orientation of rings and polygons.

source
Meshes.WindingOrientation — Type
WindingOrientation()

A method for finding the orientatino of rings and polygons based on the winding number.

TriangleOrientation()
